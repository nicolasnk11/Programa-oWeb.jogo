Instruções Detalhadas para o Jogo de Adivinhação de Números:!

Objetivo do Jogo: O objetivo do jogo é adivinhar o número secreto escolhido pelo programa. O número secreto é um número inteiro entre 1 e 100, escolhido aleatoriamente pelo programa no início do jogo.

Faixa de Números Permitidos: O usuário deve fornecer seus palpites dentro da faixa de números de 1 a 100, incluindo ambos os extremos.

Processo de Adivinhação: O usuário faz um palpite inicial digitando um número. O programa então verifica se o palpite está correto.

Feedback do Palpite: Se o palpite do usuário estiver incorreto, o programa fornecerá feedback informando se o palpite foi muito alto ou muito baixo em relação ao número secreto.

Interação: O usuário pode continuar a fazer palpites até adivinhar corretamente o número secreto.

Fim do Jogo: O jogo termina quando o usuário adivinha corretamente o número secreto. Nesse ponto, o programa exibe uma mensagem de parabéns, juntamente com o número de tentativas que o usuário levou para adivinhar o número correto.

Registro de Tentativas: Durante o jogo, o programa mantém o controle do número de tentativas que o usuário fez para adivinhar o número correto. Isso é exibido ao usuário quando ele adivinha corretamente. e que a diversão comece!
